# gtf-cli
> a simple google translate file tools


### 1.安装
* 检查是否已经安装nodejs，若未安装可前往[下载](http://nodejs.cn/download/)
  ```javascript
  // 在终端输入
  node --version
  // 若已安装会打印相应版本，如：
  // v10.13.0
  ```
* 安装翻译工具
  ```javascript
  npm install -g gtf-cli
  ```

### 2.使用
* 到需要翻译的文件目录下用终端模式打开目录
* 执行翻译命令
  ```javascript
  gtf
  ```
